## How Pobot Works

1. **Load:** Pobot reads `.md` files from this directory
2. **Answer:** Only information in this knowledge pack is shared
3. **Flag:** Uncertain or sensitive topics are marked clearly
4. **Boundary:** User questions outside this scope are respectfully declined

---

## Governance

**Core Rule:** Do not invent, infer, or soften information.  
**Safety Rule:** When in doubt, decline to answer or flag as uncertain.  
**Update Rule:** Only add content that's genuinely public-safe.
